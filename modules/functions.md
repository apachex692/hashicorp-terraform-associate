# Functions

Functions in Terraform are a powerful way to perform reusable actions in Terraform code. Following are the types of functions available in Terraform for use:

1. Numeric Functions
2. String Functions
3. Collection Functions
4. Encoding Functions
5. File-system Functions
6. Date and Time Functions
7. Hash and Crypto Functions
8. Network Functions
9. Type Conversion Functions
